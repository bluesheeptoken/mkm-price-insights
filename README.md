# MKM Trends Userscript

A small tool that injects **price histograms** and **trend insights** directly on [Cardmarket (MKM)](https://www.cardmarket.com) pages.  
Designed for **desktop and mobile**, works with Tampermonkey or as a bookmarklet. No login required.

---

## Features

- Visualize recent price history for cards
- Detect basic trends: rising, falling, stable
- Optional feedback button to submit yes/no and comments
- Mobile-friendly design

---

## Installation

### Tampermonkey

1. Install [Tampermonkey](https://www.tampermonkey.net/) for your browser.
2. Click the link below to install the userscript:

[Install MKM Trends Userscript](https://bluesheeptoken.github.io/mkm-price-insights/mkm-trends.user.js)
