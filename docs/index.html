MKM Trends Userscript

This tool injects price histograms and trend insights directly on Cardmarket (MKM) pages. Works on desktop and mobile. No login required.

Install on Tampermonkey

Click the button below to install the userscript. Once installed, open any MKM page and the histogram & trend UI will appear automatically.

Install Userscript

Feedback

After using the script, a feedback panel will appear. You can submit a quick Yes / No response, or click "Leave a comment" to give more detailed feedback.

Updates

The userscript will automatically update if you use the Tampermonkey install link. For bugs, questions, or suggestions, open an issue in this GitHub repository.
